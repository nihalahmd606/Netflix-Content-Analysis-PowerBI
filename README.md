# Netflix Content Analysis Dashboard (Power BI)

Project Overview

This project showcases an interactive **Netflix Content Analysis Dashboard** developed using **Microsoft Power BI**. The dashboard provides valuable insights into Netflix's content library by analyzing movies, TV shows, ratings, release trends, and country-wise content distribution.

It helps users explore Netflix's catalog through interactive filters and visualizations, enabling data-driven content analysis.

 Dashboard Features

 KPI Cards
- Total Titles
- Total Movies
- Total TV Shows

Interactive Visualizations
- Movies vs TV Shows Distribution
- Content Growth Over Time
- Top Countries by Content Production
- Content Rating Breakdown
- Global Content Distribution Map
  
Interactive Filters
- Rating
- Release Year
- Country

Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Data Modeling
- Data Cleaning
- Data Visualization

Key Insights

- Movies account for the majority of Netflix's content library.
- Content production has increased significantly in recent years.
- The United States contributes the highest number of titles.
- TV-MA is the most common content rating.
- Netflix content is distributed across multiple countries worldwide.
